Configuration of the Policy Enforcement

I. emayor.proprtiese file 
The file must contain the following property:

## Policy Enforcer Configurartion
emayor.pe.info.dir=conf/policies

indicateing the directory where the policy files are stored.

The policy file is an XACML policy file with the ".policy" extension.

Example of policies are located in the Policies folder.

The General.policy have to be obligatory locate in the folder.
